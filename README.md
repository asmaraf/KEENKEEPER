# KeenKeeper



## 📋 Description

KeenKeeper is a personal relationship management app designed to help you maintain meaningful connections with your friends. Keep track of interactions, set relationship goals, and nurture the friendships that matter most in your life.

## 🛠️ Technologies Used

- **Frontend Framework:** React 19
- **Language:** TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Routing:** React Router DOM
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Notifications:** Sonner
- **AI Integration:** Google Gemini API

## ✨ Key Features

### 1. Friend Management
Add and organize your friends with detailed profiles including contact information, relationship goals, and personal notes.

### 2. Interaction Tracking
Log different types of interactions (calls, texts, video calls) and maintain a timeline of all your communications.

### 3. Relationship Insights
View statistics and status indicators to identify which relationships need attention, helping you stay connected with everyone.

## 🚀 Getting Started

### Prerequisites
- Node.js (version 18 or higher)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/asmaraf/KEENKEEPER.git
   cd keenkeeper
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   Create a `.env.local` file and add your Gemini API key:
   ```
   GEMINI_API_KEY=your_api_key_here
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📜 Available Scripts

- `npm run dev` - Start the development server
- `npm run build` - Build the app for production
- `npm run preview` - Preview the production build
- `npm run lint` - Run TypeScript type checking
