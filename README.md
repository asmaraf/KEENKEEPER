# KeenKeeper

## Description

KeenKeeper is a personal relationship management app designed to help you maintain meaningful connections with your friends.

## Technologies Used

- React 19
- TypeScript
- Vite
- Tailwind CSS
- React Router DOM
- Framer Motion
- Lucide React
- Sonner
- Google Gemini API

## Key Features

1. **Friend Management**: Add and organize your friends with detailed profiles
2. **Interaction Tracking**: Log different types of interactions and maintain a timeline
3. **Relationship Insights**: View statistics and status indicators to identify relationships needing attention
